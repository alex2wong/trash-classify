# trash-classify
classify trash by [tensorflow coco-ssd](https://github.com/tensorflow/tfjs-models/tree/master/coco-ssd) model. Take a photo then classify ~~ : )

## Excuse Me， 你是什么垃圾？！
这是一款**人工智障垃圾分类app**，可以解决"你是什么垃圾"这样的灵魂拷问，谢谢~
